# Configuration

Customize Media Kit to fit your needs.

## Basic Configuration

```typescript
const media = createMedia({
  driver: myDriver,
  folders: {
    defaultFolder: 'uploads',
    enableSubfolders: true,
  },
  fileTypes: {
    allowed: ['image/*', 'application/pdf'],
    maxSize: 10 * 1024 * 1024, // 10MB
  },
});
```

## Options Reference

### `driver` (required)

Storage driver instance. S3 and GCS are included; you can also implement a custom driver.

```typescript
driver: new S3Provider({ bucket: 'my-bucket', region: 'us-east-1' })
```

See [Storage Drivers](../guides/storage-providers.mdx) for all options.

### `folders`

Organize files into folders. Folders are free-form -- there is no `baseFolders` restriction.

```typescript
folders: {
  defaultFolder: 'general',     // Default folder if not specified
  enableSubfolders: true,        // Allow nested folder paths (default: true)
  contentTypeMap: {              // Map folder patterns to content types
    'products/*': ['image/*'],
    'documents': ['application/pdf'],
  },
}
```

### `fileTypes`

Control what files can be uploaded.

```typescript
fileTypes: {
  allowed: ['image/*', 'video/*'],   // MIME patterns
  maxSize: 50 * 1024 * 1024,         // 50MB max (default: 100MB)
}
```

### `processing`

Image processing options (requires `sharp`).

```typescript
processing: {
  enabled: true,                  // Enable processing (default: true)
  quality: 80,                    // Output quality 1-100 (default: 80)
  format: 'webp',                 // Output format (default: 'webp')
  maxWidth: 2048,                 // Max width in pixels (default: 2048)
  generateAlt: true,              // Auto-generate alt text from filename
  sizes: [                        // Generate multiple size variants
    { name: 'thumbnail', width: 150, height: 150 },
    { name: 'medium', width: 800 },
    { name: 'large', width: 1600 },
    {
      name: 'hero',
      width: 1920,
      // Conditional variant -- only generated when source is wide enough
      condition: (original) => original.width >= 1920,
    },
  ],
  sharpOptions: {                 // Sharp performance tuning
    concurrency: 2,               // Concurrent processing threads
    cache: false,                  // Disable Sharp internal cache
  },
}
```

### `multiTenancy`

Isolate files by organization or tenant.

```typescript
multiTenancy: {
  enabled: true,
  field: 'organizationId',    // Field name on the document (default)
  required: false,             // Require org ID on all operations
}
```

### `deduplication`

Prevent duplicate file uploads using content hashing.

```typescript
deduplication: {
  enabled: true,               // Enable dedup (default: false)
  returnExisting: true,        // Return existing file instead of re-uploading (default: true)
  algorithm: 'sha256',         // Hash algorithm: 'md5' or 'sha256' (default)
}
```

When enabled, if a file with the same hash already exists, the existing document is returned instead of uploading a duplicate.

### `softDelete`

Enable soft deletes instead of permanent removal.

```typescript
softDelete: {
  enabled: true,               // Enable soft deletes (default: false)
  ttlDays: 30,                 // Auto-purge after N days (default: 30, 0 = never)
}
```

Soft-deleted files are marked with a `deletedAt` timestamp and excluded from queries by default. Use `media.restore(id)` to recover them, or `media.purgeDeleted()` to permanently remove expired files.

### `suppressWarnings`

Hide warnings about missing optional dependencies (like `sharp`).

```typescript
suppressWarnings: true  // Useful in production
```

## Complete Example

```typescript
import { createMedia } from '@classytic/media-kit';
import { S3Provider } from '@classytic/media-kit/providers/s3';

const media = createMedia({
  // Storage driver
  driver: new S3Provider({
    bucket: process.env.S3_BUCKET,
    region: 'us-east-1',
  }),

  // Folder organization (free-form, no baseFolders)
  folders: {
    defaultFolder: 'uploads',
    enableSubfolders: true,
  },

  // File restrictions
  fileTypes: {
    allowed: ['image/*', 'application/pdf', 'video/*'],
    maxSize: 50 * 1024 * 1024, // 50MB
  },

  // Image processing
  processing: {
    enabled: true,
    quality: 85,
    format: 'webp',
    maxWidth: 2048,
    generateAlt: true,
    sizes: [
      { name: 'thumb', width: 200, height: 200 },
      { name: 'medium', width: 800 },
      { name: 'large', width: 1600 },
      {
        name: 'hero',
        width: 1920,
        condition: (original) => original.width >= 1920,
      },
    ],
  },

  // Multi-tenancy
  multiTenancy: {
    enabled: true,
    field: 'organizationId',
    required: true,
  },

  // Deduplication
  deduplication: {
    enabled: true,
    algorithm: 'sha256',
  },

  // Soft deletes
  softDelete: {
    enabled: true,
    ttlDays: 30,
  },

  // Production mode
  suppressWarnings: true,
});
```

## Environment Variables

Store sensitive config in `.env`:

```bash
# AWS S3
AWS_ACCESS_KEY_ID=your_key
AWS_SECRET_ACCESS_KEY=your_secret
S3_BUCKET=my-bucket
S3_REGION=us-east-1

# Google Cloud
GCS_BUCKET=my-bucket
GCS_PROJECT_ID=my-project
```

Load in your app:

```typescript
import 'dotenv/config';

const media = createMedia({
  driver: new S3Provider({
    bucket: process.env.S3_BUCKET,
    region: process.env.S3_REGION,
  }),
});
```
