## Features

- Cloud storage drivers (S3, GCS, custom)
- Presigned uploads (direct browser-to-cloud)
- Auto image processing (resize, optimize, WebP/AVIF)
- Multi-size generation (thumbnails, responsive images)
- Focal-point-aware cropping
- On-the-fly asset transforms (framework-agnostic)
- File status lifecycle (pending/processing/ready/error)
- Content deduplication (SHA-256 hash)
- Tags and full-text search
- Soft deletes with auto-purge
- File replacement (same ID, new content)
- URL import
- Deep folder management (unlimited nesting)
- Alt text auto-generation
- Multi-tenant support
- Awaitable event system
- TypeScript support
- Zero config defaults
