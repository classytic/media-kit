# Media Kit Documentation

Production-grade media management for Mongoose applications.

## What is Media Kit?

A powerful library for managing file uploads in Node.js with Mongoose. Think WordPress media library, but cleaner, more flexible, and built for modern stacks.

## Key Features

- **Cloud Storage** - S3, Google Cloud Storage, or custom drivers
- **Auto Image Processing** - Generate thumbnails, resize, optimize, focal-point-aware cropping
- **Asset Transforms** - On-the-fly image resizing via URL params (framework-agnostic)
- **File Status Lifecycle** - Track uploads from pending through processing to ready
- **Tags & Search** - Tag files and search via MongoDB text index
- **Soft Deletes** - Trash and restore with configurable auto-purge
- **Content Deduplication** - SHA-256 hash-based duplicate detection
- **Multi-tenant Ready** - Organize by organization/tenant
- **Awaitable Events** - Hook into upload, delete, replace, and more
- **Presigned Uploads** - Direct browser-to-cloud uploads
- **TypeScript First** - Full type safety

## Quick Example

```typescript
import { createMedia } from '@classytic/media-kit';
import { S3Provider } from '@classytic/media-kit/providers/s3';

const media = createMedia({
  driver: new S3Provider({ bucket: 'my-uploads', region: 'us-east-1' }),
});

const file = await media.upload({
  buffer: fileBuffer,
  filename: 'photo.jpg',
  mimeType: 'image/jpeg',
  tags: ['product', 'hero'],
});

console.log(file.url);    // https://my-uploads.s3.amazonaws.com/...
console.log(file.status); // 'ready'
console.log(file.hash);   // 'sha256-...'
```
